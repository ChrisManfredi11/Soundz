1: Typography

1.1 - Change google fonts

In fonts.css:

Add your google fonts as in example bellow:


@font-face {
  font-family: 'Lato';
  font-style: normal;
  font-weight: 300;
  src: local('Lato Light'), local('Lato-Light'), url(http://themes.googleusercontent.com/static/fonts/lato/v7/KT3KS9Aol4WfR6Vas8kNcg.woff) format('woff');
}
@font-face {
    font-family: 'Lato';
    font-style: normal;
    font-weight: 400;
    src: local('Lato Regular'), local('Lato-Regular'), url(http://themes.googleusercontent.com/static/fonts/lato/v7/9k-RPmcnxYEPm8CNFsH2gg.woff) format('woff');
}
@font-face {
    font-family: 'Lato';
    font-style: normal;
    font-weight: 700;
    src: local('Lato Bold'), local('Lato-Bold'), url(http://themes.googleusercontent.com/static/fonts/lato/v7/wkfQbvfT_02e2IWO3yYueQ.woff) format('woff');
}

1.2 - SVG format fonts for smoother text in webkit browsers

In fonts.css:

Include local svg format fonts like example bellow:


@media screen and (-webkit-min-device-pixel-ratio:0) {
  @font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Light.svg') format('svg');
    font-weight: 300;
  }
}
@media screen and (-webkit-min-device-pixel-ratio:0) {
  @font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato.svg') format('svg');
    font-weight: 400;
  }
}
@media screen and (-webkit-min-device-pixel-ratio:0) {
  @font-face {
    font-family: 'Lato';
    src: url('../fonts/Lato-Bold.svg') format('svg');
    font-weight: 700;
  }
}

2: CSS/LESS

2.1 - LESS files

In "less" folder:

Style.less is the main less file and imports other css and less files


        
@import "../css/fonts.css";
@import "../css/entypo.css";
@import "../css/flaticon.css";
@import "../css/lightbox.css";
@import "animations";

2.2 - CSS files

In "css" folder:

Style.css is the main css file compiled from style.less file.
Other required css files are also loaded in header



    <link rel="stylesheet" href="assets/css/bootstrap.css">
    <link rel="stylesheet" type="text/css" href="assets/rs-plugin/css/settings.css" media="screen" />
    <link rel="stylesheet" type="text/css" href="assets/css/style.css" />

1.3 - Change colors easy

With LESS you cand change colors just by replacing one single line of code like so :


@base-color: #ec5538;

4: Javascript - jQuery library and plugins

4.1 - Include plugins

jQuery library and modernizr.custom.js are loaded in the head section.
The rest of them are loaded on the bottom of the page along with some custom settings.

See an aexample bellow:


<!-- jQuery REVOLUTION Slider  -->
    <script type="text/javascript" src="assets/rs-plugin/js/jquery.themepunch.plugins.min.js">
    <script type="text/javascript" src="assets/rs-plugin/js/jquery.themepunch.revolution.min.js">
    <script type="text/javascript"> 
        jQuery(document).ready(function($) {
            "use strict";
            jQuery('.tp-banner').revolution(
            {
                // You can edit here settings
                delay:4000,
                startwidth:1200,
                startheight:783,
                hideThumbs:10,
                fullWidth:"on",
                fullScreen:"off",
                fullScreenOffsetContainer: "",
                navigationStyle: "off",
                shadow: 0,
            });
        });
    </script>

4.2 - Audio player

For adding tracks in playlist, go to the bottom of the page and find the settings for audio player.
Learn more about jPlayer here

See the example bellow


<!-- jPlayer -->
<script type="text/javascript" src="assets/jplayer/jquery.jplayer.min.js">
<script type="text/javascript" src="assets/jplayer/jplayer.playlist.min.js">
<script type="text/javascript">

    $(document).ready(function(){
        new jPlayerPlaylist({
            jPlayer: "#jquery_jplayer_1",
            
            cssSelectorAncestor: "#jp_container_1"
        }, [
            {
                title:"Cro Magnon Man",
                mp3:"http://www.jplayer.org/audio/mp3/TSP-01-Cro_magnon_man.mp3",
                oga:"http://www.jplayer.org/audio/ogg/TSP-01-Cro_magnon_man.ogg"
            },
            {
                title:"Your Face",
                mp3:"http://www.jplayer.org/audio/mp3/TSP-05-Your_face.mp3",
                oga:"http://www.jplayer.org/audio/ogg/TSP-05-Your_face.ogg"
            },
            {
                title:"Cyber Sonnet",
                mp3:"http://www.jplayer.org/audio/mp3/TSP-07-Cybersonnet.mp3",
                oga:"http://www.jplayer.org/audio/ogg/TSP-07-Cybersonnet.ogg"
            },
        ], {
            playlistOptions: {
                autoPlay: true
            },
            swfPath: "assets/jplayer",
            supplied: "oga, mp3",
            wmode: "window",
            smoothPlayBar: false,
            keyEnabled: true
        });
    });
</script>

4.3 - Add/Remove AutoPlay for jPlayer

If you don't want the music to play when the page loads, set the AutoPlay value to False. 

playlistOptions: {
    autoPlay: true
},
